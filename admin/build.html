Materials

  • Placeholder — e.g. 6x vertical support posts (2.4m)
  • Placeholder — e.g. plywood sheets for decking
  • Placeholder — e.g. screws, brackets, cable ties
  • Placeholder — e.g. shade cloth or tarp for roof

Steps

  1. Placeholder — e.g. level the ground and lay out the footprint
  2. Placeholder — e.g. assemble the pallet base and secure together
  3. Placeholder — e.g. attach vertical supports at each corner
  4. Placeholder — e.g. lay plywood decking across the pallet base
  5. Placeholder — e.g. install overhead shade structure
  6. Placeholder — e.g. run power cables and set up PA mount points
Tip: Placeholder — e.g. orient the stage so prevailing wind carries sound toward the audience area, not into neighbouring camps.

🍳 Kitchen

Materials

  • Placeholder — e.g. folding tables (x3)
  • Placeholder — e.g. gas burners and propane tanks
  • Placeholder — e.g. shade canopy (3m x 3m)
  • Placeholder — e.g. water jerry cans (20L x4)
  • Placeholder — e.g. storage bins for dry goods

Steps

  1. Placeholder — e.g. choose a flat area downwind from sleeping zones
  2. Placeholder — e.g. set up shade canopy and stake securely
  3. Placeholder — e.g. arrange tables in an L-shape for prep and serving
  4. Placeholder — e.g. set up burners on a stable, heat-resistant surface
  5. Placeholder — e.g. organise wash station with grey water collection
Tip: Placeholder — e.g. keep propane tanks in shade and away from direct heat sources. Have a fire extinguisher within arm's reach.

Materials

  • Placeholder — e.g. aluminium poles or PVC pipe (3m x8)
  • Placeholder — e.g. shade cloth / aluminet (6m x 8m)
  • Placeholder — e.g. ratchet straps and carabiners
  • Placeholder — e.g. rebar stakes (60cm x8)
  • Placeholder — e.g. guy ropes and tensioners

Steps

  1. Placeholder — e.g. lay out the shade footprint and mark pole positions
  2. Placeholder — e.g. drive rebar stakes at each corner
  3. Placeholder — e.g. assemble poles and attach to rebar with fittings
  4. Placeholder — e.g. drape shade cloth over frame and secure with clips
  5. Placeholder — e.g. add guy ropes at 45 degrees for wind stability
Tip: Placeholder — e.g. angle the shade cloth slightly so wind passes under rather than catching it like a sail. Tighten all connections at dawn when the material is coolest.

  • Placeholder — e.g. solar shower bags (20L x3)
  • Placeholder — e.g. pallet or wooden grate for standing surface
  • Placeholder — e.g. grey water collection basin

Steps

  1. Placeholder — e.g. build the PVC frame enclosure (1m x 1m x 2m)
  2. Placeholder — e.g. attach privacy curtains on three sides
  3. Placeholder — e.g. place pallet floor inside for drainage
  4. Placeholder — e.g. hang solar shower bag from top crossbar
  5. Placeholder — e.g. set up grey water collection underneath the pallet
Tip: Placeholder — e.g. fill solar bags in the morning and leave in direct sun — they'll be warm by afternoon. Collect all grey water for proper disposal per event rules.
